# Prompt Engineering aplicado al desarrollo

En este documento explico cómo he utilizado prompts para trabajar con inteligencia artificial en el desarrollo del proyecto TaskFlow.

---

## Prompt 1

Actúa como desarrollador senior y mejora este código.

👉 Funciona porque define un rol claro.

---

## Prompt 2

Explica paso a paso qué hace este código.

👉 Ayuda a entender mejor la lógica.

---

## Prompt 3

Encuentra el error en este código y explícalo.

👉 Útil para detectar bugs.

---

## Prompt 4

Refactoriza este código para que sea más limpio.

---

## Prompt 5

Genera una función que filtre tareas completadas.

---

## Prompt 6

Optimiza este código sin cambiar su funcionalidad.

---

## Prompt 7

Escribe comentarios JSDoc para esta función.

---

## Prompt 8

Genera código basándote en este ejemplo.

---

## Prompt 9

Limita la respuesta a menos de 10 líneas.

---

## Prompt 10

Explica este código como si fuera para principiantes.

---

## Conclusión

Los mejores prompts son:
- claros
- específicos
- con contexto

Esto permite obtener mejores resultados de la IA.
